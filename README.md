# 📻 HyRadio

Reproductor de radio por internet ligero y con estética de terminal clásica. Hecho con  HTML, CSS y JS. 
Rápido y liviano.

## ✨ Características Principales

- **Estética de consola CLI**: Interfaz de texto inspirada en herramientas clásicas de terminal, con un diseño limpio y operable en una sola pantalla.
- **Gestión total de emisoras**: Agrega streams de audio mediante URL, edita nombres/enlaces y elimina estaciones directamente desde la interfaz.
- **Persistencia local**: Las emisoras, el nivel de volumen y el tema visual seleccionado se guardan automáticamente en el navegador (`localStorage`).
- **Importación y exportación JSON**: Haz copias de seguridad de las radios para transferirlas fácilmente a otros dispositivos (ideal para configuraciones *headless* o tablets dedicadas).
- **Motor de Temas Dinámico**: Incluye temas de color con paletas típicas de programación o retro.
  - *Catppuccin (Mocha, Macchiato, Frappé, Latte)*
  - *Clásicos Dev (Dracula, Nord, Tokyo Night, Monokai)*
  - *Vintage (Gruvbox, Solarized)*
  - *Monitores Retro (Fósforo Ámbar, Verde, Cian)*
- **Mobile & PWA Ready**: Incluye metaetiquetas y un favicon en SVG para funcionar como aplicación nativa a pantalla completa al guardarse en la pantalla de inicio de iOS/Android.
